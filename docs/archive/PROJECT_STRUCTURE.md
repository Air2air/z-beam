# Z-Beam Website Project Structure

This is a Next.js website for Z-Beam laser cleaning content and services.

## 📁 Project Structure

```
/
├── app/                    # Next.js application
│   ├── components/        # React components
│   ├── materials/         # Material pages
│   ├── css/              # Global styles
│   ├── utils/            # Utility functions
│   ├── layout.tsx        # Root layout
│   └── page.tsx          # Home page
├── public/                # Next.js static assets
│   └── images/           # Image assets
├── docs/                  # Documentation
├── package.json          # Next.js dependencies
├── next.config.js        # Next.js configuration
├── tailwind.config.js    # Tailwind CSS configuration
├── postcss.config.js     # PostCSS configuration
└── tsconfig.json         # TypeScript configuration
```

## 🚀 Usage

### Running the Next.js Website

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Development

```bash
# Run in development mode with hot reload
npm run dev

# Build and analyze bundle
npm run build

# Lint code
npm run lint
```

## ⚙️ Configuration

Website configuration is managed through standard Next.js files:

- **next.config.js**: Next.js application configuration
- **tailwind.config.js**: Tailwind CSS styling configuration  
- **tsconfig.json**: TypeScript compiler configuration
- **package.json**: Dependencies and build scripts

## 🏗️ Architecture

- **Next.js 13+ App Router**: Modern React framework with app directory structure
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **Component-Based**: Reusable React components
- **SEO Optimized**: Built-in SEO features and metadata management
