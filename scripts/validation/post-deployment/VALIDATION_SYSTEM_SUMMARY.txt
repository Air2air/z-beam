================================================================================
  HOW TO USE
================================================================================

AFTER EVERY DEPLOYMENT:
$ npm run postdeploy

This runs ALL 43 checks in ~5-10 minutes and reports:
  ✅ Passed: X/43
  ⚠️  Warnings: Y
  ❌ Failed: Z
  
  Success Rate: XX%
  
  Exit Code: 0 (success) or 1 (failure)

CATEGORY-SPECIFIC CHECKS:
$ npm run validate:content              # Content only
$ npm run validate:seo-infrastructure   # SEO only
$ npm run validate:performance          # Performance only
$ npm run validate:a11y                 # Accessibility only

QUICK CHECKS:
$ npm run validate:production:simple    # 2-3 minutes
$ npm run validate:production:enhanced  # 3-5 minutes
