# Partner Logos

This directory contains logo images for Z-Beam partners.

## Required Images

- `laserverse-logo.png` - Laserverse (Canada)
- `mack-laser-restoration-logo.png` - MacK Laser Restoration (Southern California)
- `netalux-logo.png` - Netalux (Belgium)

## Image Specifications

### Dimensions
- **Recommended:** 400x200px (2:1 aspect ratio)
- **Minimum:** 200x100px
- **Maximum:** 800x400px

### Format
- **Primary:** PNG with transparency
- **Alternative:** SVG (preferred for scalability)
- **Fallback:** JPG (if transparency not needed)

### File Size
- **Maximum:** 200KB per image
- **Recommended:** 50-100KB for optimal web performance

### Quality Requirements
- High resolution for retina displays
- Clean background (transparent or white)
- Centered composition
- Professional quality

## Usage

These logos are referenced in `static-pages/partners.yaml`:

```yaml
partners:
  - name: "Laserverse"
    logo: "/images/partners/laserverse-logo.png"
    url: "https://www.laserverse.ca"
```
